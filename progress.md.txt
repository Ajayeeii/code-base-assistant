# AI Codebase Assistant - Project Progress

## Project Overview

A portfolio project that allows users to clone a GitHub repository, explore its files, and interact with the codebase using a local AI model (Ollama).

---

# Tech Stack

## Frontend

* React
* Vite
* TypeScript (Strict Mode)
* Tailwind CSS
* shadcn/ui

## Backend

* FastAPI
* Python
* GitPython
* Ollama

## Database

* None (Version 1)

## Deployment

* Frontend → Vercel
* Backend → Render
